Control flow and loops

05 May 2019

Control flow is the logical order in which a computer steps through statements in a script.

An analogy could be a river. Water always flows downstream through the river, through a series of bends, and cascades.

At dam might be like a conditional statement, if the flow is low, the river might be held back, if the flow is high, the dam's spillway might open.

Further downstream might be some old ox-bows, that form an eddy or loop. The water will circulate round until released from loop and it continues to flow downstream.

The Document Object Model (DOM) and Developer Tools (DevTools)

05 May 2019

The document object model the tree of items and linkages that the browser see's when it is commanded to access a site. The browser methodically processes all of the HTML elements and their children, applies styles, runs scripts, a ultimately displays a web pages (if instructed to do so, it could just run some script and display nothing).

The DOM can be displayed from Developer Tools by right clicking and inspecting any element of a web page. It can also be accessed by writing the window. command into the browser's console.

Developer tools are there for the purpose of seeing under the hood of a webpage, and displaying how things are put together, helping to debug scripts, and many other things.
